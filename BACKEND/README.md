### Backend development

1. create a git repo
    git init

2. Add .gitignore file 

3. create .env file for environment variables and read data from .env with .env module
    npm install dotenv

4. generate package.json

6. create a express app

7. connect to database

8. Add middle wares { bodyparse, errorHandler  }

9. Designe Schemas and create models

9. define rest api for all api's
